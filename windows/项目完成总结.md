## 🚀 如何使用

### 使用 Web 版本
1. 用浏览器打开 `index.html`
2. 点击"🎬 加载示例"体验
3. 切换到"✏️ 编辑模式"创作
4. 完成！

### 打包成 Android APP

#### 方案1：在线打包（最简单）⭐
1. 压缩 `android-package` 文件夹
2. 上传到 Monaca.io 或其他服务
3. 等待构建
4. 下载 APK
5. 安装到手机

**时间**: 10-15分钟  
**难度**: ⭐（非常简单）

#### 方案2：本地构建（专业）
1. 安装 Node.js + Android Studio
2. 运行 `build.bat`（Windows）或 `build.sh`（Mac/Linux）
3. 在 Android Studio 中构建
4. 生成 APK

**时间**: 30-60分钟  
**难度**: ⭐⭐⭐（需要技术基础）

详细教程请查看：
- `android-package/README.md` - 快速开始
- `android-package/打包指南.md` - 详细教程
- `android-package/在线打包教程.md` - 在线方案
